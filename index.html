The research agent

A Python agent pipeline—not manual browsing—that searches official docs, reads auth pages, and returns structured JSON per app. Built on Composio v0.17 + Gemini with a real tool loop.

Stack

  • LLM: Gemini 2.5 Flash (OpenAI fallback supported)
  • Tools: TAVILY_SEARCH, TAVILY_EXTRACT, COMPOSIO_SEARCH_DUCK_DUCK_GO, COMPOSIO_SEARCH_FETCH_URL_CONTENT
  • SDK: Composio v0.17 with composio-google provider
  • Output: data/agent_research_results.json (incremental, resumable)

Where a human was needed

The agent is strong on explicit facts (OAuth2, API key) but weak on production gating: sandbox vs live approval, partner programs, multi-layer auth (AWS SigV4 + LWA), and "no public API" products. A human reviewed 16 high-risk apps and corrected 2 first-pass misses.

  • Prompt fix: distinguish API auth from user login
  • Manual audit on ads, fintech, enterprise data vendors
  • Honest "Not today" when docs don't exist (e.g. NotebookLM consumer API)
